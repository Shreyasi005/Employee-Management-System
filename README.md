# Employee Management System

A RESTful Employee Management System built using Java, Spring Boot, MongoDB, and Postman.

## Features
- Add Employee
- View All Employees
- View Employee By ID
- Update Employee
- Delete Employee

## Tech Stack
- Java
- Spring Boot
- MongoDB
- Maven
- REST APIs
- Postman

## Project Architecture
Controller → Service → Repository → MongoDB

## API Endpoints

POST /employees
GET /employees
GET /employees/{id}
PUT /employees/{id}
DELETE /employees/{id}
